# 🌌 Lumina - Premium Wallpaper App

Lumina is a highly expressive, premium wallpaper discovery web application built with React, Vite, and the Pexels API. 

## ✨ Features
- **Expressive Glassmorphism UI**: Beautiful dark mode interface with a sleek light mode toggle.
- **Search by Color**: Filter wallpapers by exact color tones using the interactive color palette.
- **Framer Motion Animations**: Buttery smooth physics-based layout transitions and hover effects.
- **Infinite Scrolling**: Never run out of beautiful imagery.
- **Instant Downloads**: Save high-resolution wallpapers directly to your device with one click.

## 🚀 Tech Stack
- **Framework**: React (via Vite)
- **Styling**: Vanilla CSS & CSS Variables
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Notifications**: React Hot Toast
- **Data Source**: Pexels API

## 🛠️ Local Setup
1. Clone the repository: `git clone https://github.com/rehmansaini462-eng/lumina-wallpapers.git`
2. Install dependencies: `npm install`
3. Create a `.env` file in the root directory and add your Pexels API Key:
   ```env
   VITE_PEXELS_API_KEY=your_api_key_here
   ```
4. Start the development server: `npm run dev`
